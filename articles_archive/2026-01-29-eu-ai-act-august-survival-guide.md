**2026年8月、AI業界が変わる。**

EU AI法（AI Act）の**高リスクAI規制が完全施行**されます。

:::note alert
違反した場合の罰金：**最大3,500万ユーロ（約56億円）**または**全世界売上の7%**
:::

「うちはEUでビジネスしてないから関係ない」

**本当にそうですか？**

## EU AI法の基本

### リスクベースの4段階規制

```
┌─────────────────────────────────────────────────────────┐
│              EU AI法のリスク分類                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  🚫 禁止（Unacceptable Risk）                           │
│  ├─ 社会信用スコアリング                                 │
│  ├─ リアルタイム顔認識（公共空間）                        │
│  └─ 感情認識（職場・学校）                               │
│                                                         │
│  ⚠️ 高リスク（High Risk）← 8月に規制強化                │
│  ├─ 採用・人事AI                                        │
│  ├─ 信用スコアリング                                    │
│  ├─ 教育評価AI                                          │
│  └─ 法執行AI                                            │
│                                                         │
│  📝 限定リスク（Limited Risk）                          │
│  └─ チャットボット（AI開示義務のみ）                     │
│                                                         │
│  ✅ 最小リスク（Minimal Risk）                          │
│  └─ ほとんどのAI（規制なし）                             │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 施行スケジュール

| 時期 | 内容 |
|------|------|
| 2024年8月 | AI法発効 |
| 2025年2月 | 禁止AI規制開始 |
| **2026年8月** | **高リスクAI規制完全施行** |
| 2027年8月 | すべての規制完全適用 |

## 日本企業が影響を受けるケース

### 直接的な影響

```
こんな場合はEU AI法の対象
────────────────────────────────────────
✅ EU市民のデータを処理するAI
✅ EU市場向けのAI製品・サービス
✅ EU企業と取引がある場合
✅ グローバルSaaSを提供している場合
────────────────────────────────────────
```

### 具体例

| 業種 | 影響を受けるケース |
|------|-------------------|
| **HR Tech** | EU子会社の採用にAIを使用 |
| **金融** | EU居住者への与信判断AI |
| **教育** | EU向けオンライン学習プラットフォーム |
| **SaaS** | グローバル展開しているB2B AI |

## 高リスクAIの義務

### 8月から必須になること

| 義務 | 内容 |
|------|------|
| **リスク管理システム** | AI のリスクを継続的に評価・軽減 |
| **データガバナンス** | 訓練データの品質・バイアス管理 |
| **技術文書** | AI の仕組みを詳細に文書化 |
| **ログ記録** | AI の判断を追跡可能に |
| **透明性** | ユーザーへの説明義務 |
| **人間の監視** | AI 判断を人間が監視・介入可能に |
| **精度・セキュリティ** | 一定水準の性能保証 |

### 違反時の罰則

| 違反内容 | 罰金 |
|---------|------|
| 禁止AIの使用 | 売上の**7%** or **€35M** |
| 高リスクAI義務違反 | 売上の**3%** or **€15M** |
| 虚偽情報提供 | 売上の**1%** or **€7.5M** |

## EU AI法は「延期」されるのか？

### 最新の動向

> 欧州委員会は、業界からの圧力と準備不足を理由に、**高リスク義務を1年延期**することを検討中。

しかし：

:::note warn
**延期は確定ではない。** 8月施行を前提に準備を進めるべき。
:::
