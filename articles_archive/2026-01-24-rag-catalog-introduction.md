## RAG開発、ツール選びで時間溶けてませんか？

「RAGアプリを作りたい」と思ったとき、最初にぶつかる壁。

**「どのフレームワーク使えばいいの？」**
**「ベクトルDBって何種類あるの？」**
**「LangChainとLlamaIndex、どっちがいい？」**

Google検索しても、古い情報や偏った比較記事ばかり。GitHubを彷徨っても、スター数だけじゃ判断できない。

結局、ツール選定だけで何時間も溶ける...。

## そこで作りました：RAG カタログ

https://rag-catalog.vercel.app

**45以上のRAG関連ツール**を、カテゴリー別に整理したカタログサイトです。

### 8つのカテゴリーで整理

| カテゴリー | 主なツール |
|-----------|-----------|
| フレームワーク | LangChain, LlamaIndex, Haystack |
| ベクトルDB | Pinecone, Milvus, Chroma, Weaviate |
| エンベディング | OpenAI, Cohere, sentence-transformers |
| 評価ツール | RAGAS, Langfuse, Phoenix |
| チャンキング | Unstructured, LangChain Splitters |
| リランキング | Cohere Rerank, bge-reranker |
| データローダー | LlamaHub, Unstructured |
| オーケストレーション | Flowise, Langflow |

## こんな情報が一目でわかる

### GitHub Stars で人気度チェック

「みんなが使ってる」は重要な指標。各ツールのGitHub Stars数を表示しているので、コミュニティの活発さがすぐわかります。

**今週のHOTツール**も一目瞭然。トレンドを逃しません。

### ライセンス確認

商用利用できる？MIT？Apache 2.0？

プロダクション投入前に確認必須のライセンス情報も掲載。

### タグで絞り込み

- Python / TypeScript
- オープンソース / マネージド
- セルフホスト可能
- エンタープライズ向け

自分のスタックに合ったツールを素早く見つけられます。

## 使い方

### 1. カテゴリーから探す

「ベクトルDBを探してる」→ [ベクトルDBカテゴリー](https://rag-catalog.vercel.app/category/vectordb)へ

スター数順にソートされているので、人気ツールがすぐ見つかります。

### 2. 検索で探す

「Pythonで使えるRAGフレームワーク」→ 検索窓に「Python framework」

ツール名、説明文、タグを横断検索。

### 3. 詳細ページで比較

各ツールの詳細ページでは：
- 機能の特徴
- 公式サイト・GitHubリンク
- 同カテゴリーの人気ツール

を確認できます。

## なぜ作ったのか

私自身、RAGアプリを開発するとき、毎回ツール選定で苦労していました。

- 「Pinecone高いな...代替ないの？」
- 「LangChainのチャンキング機能、他にいいのないの？」
- 「評価どうやるの？ツールあるの？」

**必要なときにサッと調べられる場所**が欲しかった。

だから作りました。
