## Claude 5で変わること

### 開発者への影響

```python
# Claude 3.5の限界
response = claude.messages.create(
    model="claude-3-5-sonnet",
    max_tokens=8000,  # 出力制限
    # 動画は処理できない
    # 長時間タスクは不安定
)

# Claude 5で可能になること（予想）
response = claude.messages.create(
    model="claude-5",
    max_tokens=100000,  # 大幅拡張
    media=[video_file, audio_file],  # マルチモーダル
    task_duration="4 hours",  # 長時間タスク
    memory_id="project-abc",  # 永続メモリ
)
```
