## テキストしか返せないAI、まだ使ってるの？

「AIに質問したら、長文のテキストがダーッと返ってきて、読むのがめんどくさい...」

そんな経験、ありませんか？

**2026年1月26日、その常識が完全に覆されました。**

AnthropicとOpenAIが共同で発表した**MCP Apps**により、AIがチャット内で**ダッシュボード、フォーム、グラフ、3Dビジュアライゼーション**を直接表示できるようになったのです。

:::message
結論から言うと、MCP Appsは「AIとの対話」を「AIとの協働作業」に変える革命的なアップデートです。Claude、ChatGPT、VS Codeなど主要クライアントで既に利用可能です。
:::

## MCP Appsとは？30秒で理解する

**MCP（Model Context Protocol）**は、AIが外部ツールやデータにアクセスするための「USB-Cのような」標準規格です。

これまでのMCP：
- ✅ データベースからデータを取得
- ✅ APIを叩いて情報を取得
- ✅ ファイルを読み書き
- ❌ **UIの表示は不可能**

**MCP Apps**により追加された機能：
- ✅ インタラクティブなダッシュボード表示
- ✅ フォーム入力と送信
- ✅ リアルタイムグラフ更新
- ✅ 3Dビジュアライゼーション
- ✅ PDFビューア
- ✅ インタラクティブマップ

## 【衝撃】実際に何ができるのか？5つの実用例

### 1. プロジェクト管理がチャット内で完結

「来週のスプリント計画を見せて」とAIに聞くと...

**Before（従来）:**
```
タスク1: ログイン機能実装 - 担当: 田中 - 期限: 1/30
タスク2: API設計 - 担当: 鈴木 - 期限: 1/31
タスク3: テスト作成 - 担当: 佐藤 - 期限: 2/1
...（延々とテキスト）
```

**After（MCP Apps）:**
→ **Asanaのタイムラインがチャット内に表示**
→ ドラッグ＆ドロップで期限変更
→ 担当者の再割り当ても可能
→ 別タブを開く必要なし！

### 2. データ分析がインタラクティブに

```
あなた: 「先月の売上データを可視化して」
```

従来: CSVファイルが添付される → Excelで開く → グラフ作成

**MCP Apps**: **インタラクティブなグラフがその場で表示** → クリックでドリルダウン → フィルタリングも可能

### 3. PDFレビューがチャット内で

「この契約書の問題点をハイライトして」

→ PDFビューアがチャット内に表示
→ AIがハイライトした箇所を直接確認
→ コメント追加も可能

### 4. 3Dモデルの確認

建築設計やプロダクトデザインで：

「この3Dモデルを360度回転させて確認したい」

→ Three.jsベースの3Dビューアがチャット内に表示
→ マウスで回転・ズーム可能

### 5. リアルタイムモニタリング

「サーバーの状態をモニタリングして」

→ リアルタイムダッシュボードが表示
→ CPU使用率、メモリ、エラーログがライブ更新

## 技術的な仕組み（開発者向け）

MCP Appsは以下の技術スタックで構成されています：

```
┌─────────────────────────────────────────┐
│           MCP Client (Claude等)          │
├─────────────────────────────────────────┤
│         Sandboxed iframe                 │
│    ┌─────────────────────────────┐      │
│    │   MCP App (React/HTML)      │      │
│    │   - ui:// URI scheme        │      │
│    │   - JSON-RPC通信            │      │
│    └─────────────────────────────┘      │
├─────────────────────────────────────────┤
│           MCP Server                     │
└─────────────────────────────────────────┘
```

### セキュリティモデル

「他人が作ったUIを表示するのは危険では？」

MCP Appsは4層のセキュリティで保護されています：

1. **iframe sandboxing**: 全UIがサンドボックス化されたiframe内で実行
2. **事前テンプレート審査**: ホストがHTMLコンテンツを事前にレビュー可能
3. **監査可能なメッセージ**: 全通信がJSON-RPCでログ可能
4. **ユーザー同意**: UI起動のツール呼び出しに明示的な承認が必要

## 今すぐ試す方法

### Claude Desktop（最も簡単）

1. Claude Desktopを最新版にアップデート
2. 設定 → MCP → 既存のMCPサーバーがMCP Apps対応していれば自動的に利用可能

### 開発者として実装する

```typescript
// @mcp-ui/server パッケージを使用
import { McpServer } from '@mcp-ui/server';

const server = new McpServer({
  name: 'my-dashboard-app',
  version: '1.0.0',
});

server.tool('show_dashboard', {
  description: 'インタラクティブなダッシュボードを表示',
  inputSchema: { type: 'object', properties: {} },
  handler: async () => {
    return {
      type: 'ui',
      uri: 'ui://dashboard',
      html: `
        <div id="dashboard">
          <h1>売上ダッシュボード</h1>
          <canvas id="chart"></canvas>
        </div>
        <script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
        <script>
          // Chart.jsでグラフを描画
          new Chart(document.getElementById('chart'), {
            type: 'bar',
            data: { /* ... */ }
          });
        </script>
      `
    };
  }
});
```

## AnthropicとOpenAIが「協力」した理由

「え、ライバル企業が協力？」

そうなんです。MCP Appsは**Anthropic、OpenAI、MCP-UI**の3者が共同で策定しました。

理由は明確：
