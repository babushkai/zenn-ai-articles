# Zenn.dev AI Tech Articles

AI・LLMエージェント、Claude Code等の最新技術に関する日本語技術記事を自動生成・公開するリポジトリです。

## 概要

このリポジトリはZenn.devとGitHub連携しており、`articles/`ディレクトリにマークダウン記事をpushすると自動的にZenn.devに公開されます。

## 記事のトピック

- 🤖 AIエージェント / マルチエージェントシステム
- 🧠 LLMエージェント
- 💻 Claude Code テクニック
- 🔌 MCP (Model Context Protocol)
- 📝 プロンプトエンジニアリング
- 📚 RAG システム

## ローカルプレビュー

```bash
npx zenn preview
```

http://localhost:8000 でプレビュー確認できます。
